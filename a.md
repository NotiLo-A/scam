Ниже — краткая записка-инструкция по запуску и использованию программы распознавания лиц, реализованной только на cv2, без установки дополнительных библиотек.


---

Инструкция по использованию программы распознавания лиц

1. Назначение программы

Программа предназначена для обнаружения и идентификации лиц в реальном времени по видеопотоку с веб-камеры, а также для добавления (регистрации) новых лиц без перезапуска приложения.

Реализация выполнена исключительно с использованием стандартных средств библиотеки OpenCV (cv2).


---

2. Требования к системе

Python 3.x

Установленная библиотека opencv-python

Подключённая веб-камера


Дополнительные библиотеки и модели не требуются.


---

3. Подготовка к запуску

1. Убедитесь, что файл программы (например face_recognition_live_enroll.py) находится в рабочей директории.


2. В той же директории автоматически будет создана папка:

faces/

В ней хранится база данных лиц.




---

4. Запуск программы

Запуск осуществляется стандартной командой:

python face_recognition_live_enroll.py

После запуска откроется окно с изображением с веб-камеры.


---

5. Работа программы

Программа работает одновременно в двух режимах:

распознавание лиц;

добавление (сохранение) новых лиц.


Переключение режимов не требуется.


---

6. Добавление (регистрация) нового лица

6.1 Ввод имени

Активное поле ввода отображается внизу окна:

Name: <имя>

Введите имя пользователя с клавиатуры.

Поддерживаются латинские буквы, цифры и символы.

Удаление символа — клавиша Backspace.


6.2 Сохранение лица

1. Убедитесь, что лицо чётко видно в камере.


2. Нажмите левой кнопкой мыши на кнопку SAVE.


3. Программа автоматически сохранит серию изображений лица.


4. В процессе отображается счётчик:

Saving: N / SAMPLES


5. После завершения сохранения лицо добавляется в базу и сразу начинает распознаваться.



❗ Перезапуск программы не требуется.


---

7. Распознавание лиц

При появлении лица в кадре программа пытается сопоставить его с базой.

Над лицом отображается:

имя пользователя — при успешном совпадении;

Unknown — если лицо не найдено в базе.



Распознавание происходит в реальном времени.


---

8. Управление

Действие	Клавиша / действие

Ввод имени	Клавиатура
Удалить символ	Backspace
Сохранить лицо	Клик по кнопке SAVE
Выход из программы	Esc



---

9. Хранение данных

Структура базы данных:

faces/
├── ivan/
│   ├── 0.png
│   ├── 1.png
│   └── ...
├── alex/
│   ├── 0.png
│   └── ...

Каждая папка соответствует одному пользователю.

Удаление папки полностью удаляет пользователя из системы.



---

10. Ограничения

Устойчивость ниже, чем у нейросетевых решений

Чувствительность к освещению и ракурсу

Не предназначено для промышленной биометрии


Ограничения обусловлены требованием использовать только стандартные средства cv2.


---

11. Завершение работы

Для корректного завершения работы нажмите клавишу Esc в окне программы.
